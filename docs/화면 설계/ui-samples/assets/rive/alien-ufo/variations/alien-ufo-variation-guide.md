# Alien UFO Variation Guide

## Scope

기존 `alien-ufo-source.svg`의 비율, UFO shape, dome scale, alien head/body ratio를 유지한 variation package야.

## Review Sheets

| File | Purpose |
|---|---|
| `alien-ufo-expression-sheet.png` | 12개 표정 전체 preview |
| `alien-ufo-direction-sheet.png` | 8방향 reference preview |
| `alien-ufo-expression-sheet.svg` | expression sheet source |
| `alien-ufo-direction-sheet.svg` | direction sheet source |
| `alien-ufo-variation-manifest.json` | variation file manifest |

## Notes

- 모든 개별 variation은 512x512 transparent SVG야.
- expression variation은 기존 front source를 기준으로 눈, 입, cheek, small accessory만 바꿨어.
- direction variation은 2D source 기반 turn reference야. 실제 Rive/3D turntable에서 완전한 side/back rig가 필요하면 이 sheet를 기준으로 side/back source를 다시 레이어링하는 게 좋아.
- 원본 캐릭터 권리 상태가 명확하지 않으면 production 적용 전 권리 확인 필요.

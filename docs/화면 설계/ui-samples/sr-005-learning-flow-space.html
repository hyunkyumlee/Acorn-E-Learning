개념 학습 · JAVA 반복문

반복문은 같은 작업을 조건에 맞춰 반복하는 구조입니다.

반복문을 쓰면 같은 코드를 여러 번 작성하지 않고, 조건이 참인 동안 같은 흐름을 유지할 수 있습니다.

for (int i = 1; i <= 3; i++) {
  System.out.println(
    "orbit " + i
  );
}

핵심은 시작값, 반복 조건, 증가 규칙입니다. 이 세 가지를 읽으면 반복이 몇 번 실행되는지 예측할 수 있습니다.

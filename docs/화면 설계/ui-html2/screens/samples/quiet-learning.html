현재 개념

조건식은 true 또는 false로 평가되고, 실행 블록은 첫 번째 일치 조건에서 결정된다.

int score = 84;

if (score >= 90) {
  System.out.println("A");
} else if (score >= 80) {
  System.out.println("B");
} else {
  System.out.println("retry");
}

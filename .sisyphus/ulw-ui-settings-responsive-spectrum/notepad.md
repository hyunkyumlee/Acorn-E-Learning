# ULW Notepad: Settings / Responsive / Spectrum / Design Spec

## Goal
- 설정 화면을 Figma export 레이아웃 기준으로 디자인 요소 적용.
- 분석, 커뮤니티, 마이페이지 반응형 전환 폭을 학습 화면과 같은 기준으로 통일.
- 라이트모드 gradient/spectrum 색 변화 폭 축소.
- global CSS, MD, Notion `디자인 명세` 페이지 동기화.

## Evidence Targets
- before: settings / analysis / community / mypage at 820px and 1024px.
- after: same screens and widths, plus light spectrum sanity check.

## Decisions
- Mobile breakpoint target follows learning screen: hard mobile under 760px, tablet adaptations in 761-1180px range.
- Light spectrum should retain energetic cyan-mint identity but reduce cyan-to-lime/yellow drift.
